# my-nix-modules

This is a personal collection of NixOS and Home Manager modules. Its sole purpose is to share some ideas and patterns — it is **not** intended to be used as a dependency or in production by anyone but me.

Feel free to browse for inspiration, but expect things to change or break without notice.
